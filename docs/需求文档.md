# 需求

目前需要处理指定目录下所有的笔记（可能有多级目录结构，保持目录结构不变），千级笔记,大部分为md文档，部分为pdf、html。
需要：
1. 统一转换为本地md文档
2. 内容净化
两个功能可以单独使用，也可以串联使用

本地环境
win11
python3、pip

## 内容净化

编写python3代码实现对指定目录下剪藏的博客进行广告清理，md格式，广告可能是开头、末尾、或者在中间插入，可能出现多次。
高性能，稳定易用，并且清理时候不影响正常内容

常见广告特征：

    包含常见广告词：广告、推广、赞助、推荐阅读、你可能喜欢、热门文章等。

    特定链接域名：如amazon、taobao、jd等推广链接。

    特定HTML注释标记：如<!-- ad -->。

    重复的段落。
